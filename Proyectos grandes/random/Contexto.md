# Contexto: Random Hub (Funciones Extremadamente Aleatorias)

Este directorio contiene 5 funciones experimentales diseñadas para confundir, divertir y asombrar al usuario con efectos visuales y lógicos de gran escala.

## Funciones Incluidas:
1.  **💻 Código Matrix**: Una lluvia de caracteres estadísticos verdes que cubre toda la pantalla (Efecto clásico).
2.  **🐱 Invasión Mascota**: Decenas de emojis de animales que rebotan por todas partes como si la pantalla fuera un salvapantallas antiguo.
3.  **🪞 Mundo Espejo**: Invierte la pantalla horizontalmente y aplica un filtro de colores negativos. ¡Todo se vuelve opuesto!
4.  **🍮 UI Gelatina**: Hace que todos los botones y paneles vibren y se deformen como gelatina cuando pasas el ratón sobre ellos.
5.  **💥 Explosión Datos**: Los números de tu dataset actual salen disparados desde tu cursor cada vez que lo mueves, creando una estela numérica.

## Estructura Técnica:
- `js/random-hub.js`: El cerebro que coordina el menú y las activaciones.
- `js/matrix-rain.js`: Sistema de canvas para la lluvia digital.
- `js/mascot-invasion.js`: Motor de partículas de emojis.
- `js/mirror-world.js`: Manipulación directa de filtros CSS en el elemento raíz.
- `js/jelly-ui.js`: Inyección dinámica de animaciones de deformación elástica.
- `js/data-explosion.js`: Generador de estelas numéricas vinculadas al dataset activo.
